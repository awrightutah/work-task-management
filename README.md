# Work Task Management Platform

A unified customer follow-up and task management platform for the Customer Loyalty SUPLINE department.

## Features
- Case Management (CA, ESC, S-xxx tracking)
- Follow-Up Manager with automated reminders
- Microsoft Teams Calendar Integration
- Salesforce CRM Contact Sync
- Admin Dashboard with team oversight
- Escalation Management with SLA tracking

## Getting Started

### Local Development
```bash
npm install
npm start
```
Then open http://localhost:3000

### Login
- **Agent/User role** — standard agent view
- **Administrator role** — full team oversight, all calendars, reports

## Deployment
This app is configured for Railway deployment.
Set the `PORT` environment variable if needed (Railway sets this automatically).
